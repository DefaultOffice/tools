Pinterest Board Downloader

Download all images from a Pinterest board via Terminal

Terminal Command
Enter a URL above to generate command

How to use

First time setup Run this once in Terminal to install the download tool:
pip3 install gallery-dl
1

Paste your Pinterest board URL above

2

Click Generate Command then Copy

3

Open Terminal (Applications → Utilities → Terminal)

4

Paste the command (⌘V) and press Enter

5

Images will download to ~/Downloads/Pinterest/
